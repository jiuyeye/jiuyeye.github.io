学习React15.0的源码。主要包括addons、isomorphic、renderers、shared、core、test几部分。
<!-- more -->
### addons:包含一系列的工具方法插件，

### isomorphic：包含一系列同构方法
### shared：包含一些公用或常用方法
### test：包含一些测试方法；
### core/tests：包含一些边界错误的测试用例
### renderers：React的核心部分，包含了大部分功能实现
### dom：包含client、server和shared。
